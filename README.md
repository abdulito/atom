Maker
====

Maker is a simple and powerful OO toolkit for Javascript

Installing Maker
----------

Using npm 
```
$ cd <your-app>
% npm install maker
```

From git:
```
% git clone git@github.com:objectlabs/maker.git
$ cd <your-app>
% npm install <path-to-maker>
```
To run unit tests
-----------------

```node
% node ./test/all.js
```
